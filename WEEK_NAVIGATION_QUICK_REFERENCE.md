# Week Navigation - Quick Reference

## Keyboard Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| Previous Week | `⌘ + ←` | Go back one week |
| Next Week | `⌘ + →` | Go forward one week |
| This Week | `⌘ + T` | Jump to current week |
| Refresh | `⌘ + R` | Refresh calendar data |

## Navigation Buttons

### Previous Week (`←`)
- Click the left chevron button
- Use keyboard: `⌘ + ←`
- Moves to the week before the current view

### Next Week (`→`)
- Click the right chevron button
- Use keyboard: `⌘ + →`
- Moves to the week after the current view

### This Week Button
- Click the "Today" button
- Use keyboard: `⌘ + T`
- Instantly returns to the current week
- Highlights with accent color

### Refresh Button
- Click the circular arrow button
- Use keyboard: `⌘ + R`
- Manually syncs calendar events
- Shows spinning animation during refresh

## Week Display Format

The week range is displayed in the header as:
```
Mon, 1 Jan – Sun, 7 Jan
```

Shows:
- Day of week abbreviation (Mon, Tue, etc.)
- Day number
- Month abbreviation
- Complete week range from Monday to Sunday

## Tips

1. **Quick Navigation**: Use keyboard shortcuts for fastest navigation
2. **Return Home**: Press `⌘ + T` anytime to jump back to current week
3. **Visual Feedback**: Watch for hover effects to confirm button interaction
4. **Multiple Views**: Shortcuts work in Day, Week, Month, and Year views
5. **Continuous Navigation**: Hold and click buttons repeatedly for quick multi-week jumps

## Common Workflows

### Navigate to Future Week
1. Press `⌘ + →` multiple times
2. Or click Next Week button repeatedly

### Navigate to Past Week
1. Press `⌘ + ←` multiple times
2. Or click Previous Week button repeatedly

### Find Today's Week
1. Press `⌘ + T` once
2. Or click "Today" button

### Sync Latest Events
1. Press `⌘ + R`
2. Or click refresh button
3. Wait for spinning animation to complete

## Accessibility

- All buttons support VoiceOver
- Keyboard navigation fully supported
- Visual indicators for all states
- Clear button labels
- High contrast on hover

## Troubleshooting

**Shortcuts not working?**
- Ensure Calendar page is focused
- Check system keyboard settings
- Verify no conflicting shortcuts

**Week range incorrect?**
- Check system calendar settings
- Verify time zone is correct
- Restart app if needed

**Refresh not working?**
- Check internet connection
- Verify calendar permissions
- Try manual app restart
