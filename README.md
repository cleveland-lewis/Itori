# Roots

Roots is an academic planning and productivity application for macOS. It provides structured task management, automated scheduling, and time-based planning tools designed for complex academic workloads.

## Overview

Roots centralizes course management, assignment tracking, and study scheduling into a unified interface. The application processes academic tasks and generates time-blocked schedules based on due dates, priority levels, and available time slots.

## Features

- **Course Management** – Organize academic courses, track enrollment periods, and associate assignments with course structures
- **Assignment Tracking** – Categorized task management for exams, quizzes, projects, homework, and reading assignments
- **Automated Scheduling** – Time-based planner that distributes work sessions across available time blocks
- **Calendar Integration** – Visual timeline showing scheduled sessions and assignment deadlines
- **Grade Tracking** – Basic grade calculation and academic performance monitoring
- **Focus Tools** – Timer-based productivity features with session management
- **Practice Testing** – Question-based review system for exam preparation

## Platform Support

- macOS 13.0 or later
- iOS 16.0 or later (companion interface)
- iPadOS 16.0 or later (companion interface)
- watchOS 9.0 or later (limited feature set)
