# French Translation Review

**Language Code**: fr
**Strings**: 1180

## How to Use This

### Option 1: GitHub Copilot Chat
1. Open `review_fr.txt` in VS Code
2. Select all text (Cmd+A)
3. Open Copilot Chat (Cmd+Shift+I)
4. Paste the prompt and send
5. Copilot will analyze based on its knowledge of French from Reddit/forums

### Option 2: ChatGPT
1. Go to chat.openai.com
2. Copy-paste the prompt from `review_fr.txt`
3. ChatGPT will review using its vast French training data

### Option 3: Claude (Best Quality)
1. Go to claude.ai
2. Upload `review_fr.txt` or paste the prompt
3. Claude will provide detailed cultural analysis

## What to Expect

The AI will identify translations that:
- Sound too formal or too casual
- Use outdated terminology
- Feel "machine translated"
- Don't match how students actually talk

## Context

**Reddit/Forum Context**: r/france, francophone communities
**Key Considerations**: Informal "tu" form, modern French, avoid anglicisms
