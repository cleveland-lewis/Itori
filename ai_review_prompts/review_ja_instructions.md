# Japanese Translation Review

**Language Code**: ja
**Strings**: 1191

## How to Use This

### Option 1: GitHub Copilot Chat
1. Open `review_ja.txt` in VS Code
2. Select all text (Cmd+A)
3. Open Copilot Chat (Cmd+Shift+I)
4. Paste the prompt and send
5. Copilot will analyze based on its knowledge of Japanese from Reddit/forums

### Option 2: ChatGPT
1. Go to chat.openai.com
2. Copy-paste the prompt from `review_ja.txt`
3. ChatGPT will review using its vast Japanese training data

### Option 3: Claude (Best Quality)
1. Go to claude.ai
2. Upload `review_ja.txt` or paste the prompt
3. Claude will provide detailed cultural analysis

## What to Expect

The AI will identify translations that:
- Sound too formal or too casual
- Use outdated terminology
- Feel "machine translated"
- Don't match how students actually talk

## Context

**Reddit/Forum Context**: r/LearnJapanese, r/japan, anime/tech communities
**Key Considerations**: Politeness levels, katakana vs kanji, modern youth language
